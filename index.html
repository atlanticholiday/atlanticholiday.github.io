      Rollout

      Time Clock Go-Live Checklist

      Use this before switching the team to the live clock.

      1. Create one dedicated tablet login and assign it the time-clock-station role.
      2. Keep each colleague on their own access login or linked employee record. The tablet account is shared only for the station picker.
      3. Sign in on the tablet with the station account and confirm it opens the shared colleague picker and defaults to Portuguese.
      4. Test one full attendance cycle on the tablet: clock in, start break, end break, and clock out.
      5. Confirm the screen returns to the colleague list after a saved punch and that search still finds colleagues quickly.
      6. As a manager, verify the attendance print view, weekly/monthly switching, and one manual correction with a valid reason.
      7. Toggle the browser offline and online once to confirm the sync banner changes and punches recover correctly after reconnecting.
      8. Do a live pilot with one or two trusted colleagues before the full rollout.
      Important: the tablet account should stay signed in on the shared device. Colleagues should only pick their name and save punches there; they should not sign in and out with personal passwords on the tablet.

        Airbnb VAT Invoices
        Run the local invoice exporter without menu clicking

        Local automation

        Export Airbnb reservation VAT invoices

        This page points to the local Node script in scripts/airbnb/. The exporter reads Airbnb's own Completed Reservations payload, opens each invoice URL directly, and saves PDFs without clicking the three-dot menu or the browser print dialog.

        Before You Run It

        Sign into Airbnb in the opened browser, go to Completed Reservations, and apply the exact filters or date range you want before pressing Enter in the terminal.

        What It Saves

        PDFs are saved into %USERPROFILE%\Downloads\airbnb-reservation-invoices\. The reservation exporter does not write manifest files anymore.

        Terminal Note

        A normal browser page cannot launch a local PowerShell command directly, so this page gives you copy buttons instead.

        Command options

        Pick the behavior you want, then copy the generated command. Assume the terminal is already running in this repo root.

        Run mode

        Generated command

        Opens the Airbnb profile browser if needed, waits for you to prepare the list, then exports the invoices to PDF. 

        
                        
        
                            Current working directory: this repo root, so the command does not need a cd prefix.
